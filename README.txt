this program's major purpose is to check and record football goals in live matches.

you can install requirements in Auto_goal_record_and_detection_on_live_football_matches folder with:
    pip install -r requirements.txt

    also Firefox and geckodriver is needed you can download them from:
        Firefox: https://www.mozilla.org/en-US/firefox/new/
        geckodriver: https://github.com/mozilla/geckodriver/releases

and you can use commands below to run it:
    Linux/MacOs:
        python -m GoalRecorder

    Windows:
        py -m GoalRecorder

    also you can run it in PyCharm or etc by click the green arrow

in addition, you can learn its features by enter help in the menu
